# Summary

**Conceptual Summary**

This organogram visualizes a speculative spherical instrument integrating microphone and speaker elements. The central sphere (white circle) represents a resonant acoustical cavity, symbolizing an omnidirectional sound source with uniform wave propagation. A small rhombus (orange) denotes the microphone as an input interface, while a horizontal cone (cyan) indicates the speaker as an output interface. Arrows show the signal flow: sound enters via the mic, is processed within the spherical resonator (implied internal transformation), and exits amplified through the speaker. The performer (small white circle with dotted line) interacts from a distance, suggesting non-contact control. Colors follow organological conventions: orange for electronic input, cyan for output, white for resonance. Acoustically, the sphere provides isotropic resonance, while the electronic components enable feedback loops or spatial sound modulation, merging acoustic resonance with electroacoustic transduction in a compact, performative system.

import matplotlib.pyplot as plt
import numpy as np
from matplotlib.patches import Circle, Polygon, FancyArrowPatch
from matplotlib.lines import Line2D

fig, ax = plt.subplots(figsize=(8, 8))
ax.set_xlim(-1.5, 1.5)
ax.set_ylim(-1.5, 1.5)
ax.set_aspect('equal')
ax.axis('off')
fig.patch.set_alpha(0.0)

# Main resonant sphere (acoustical shape)
sphere = Circle((0, 0), 1, fill=False, edgecolor='white', linewidth=2.5, alpha=0.9)
ax.add_patch(sphere)

# Microphone (input interface) - small rhombus
mic = Polygon([[0.3, 1], [1, 0.3], [0.3, 0], [0, 0.3]], closed=True,
              facecolor='orange', edgecolor='darkorange', linewidth=1.5)
ax.add_patch(mic)

# Speaker (output interface) - horizontal cone
speaker = Polygon([[0.3, 1], [1, 0], [0, 0]], closed=True,
                  facecolor='cyan', edgecolor='darkcyan', linewidth=1.5)
speaker.set_transform(ax.transData + plt.matplotlib.transforms.Affine2D().rotate_deg(180).translate(-1.2, -0.5))
ax.add_patch(speaker)

# Arrows for signal flow
arrow1 = FancyArrowPatch((0.65, 0.65), (0.15, 0.15),

                         arrowstyle='->', color='lime', linewidth=2, mutation_scale=20)
